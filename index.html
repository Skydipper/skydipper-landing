VIZZUALITY llevará a cabo un proyecto estratégico de desarrollo de una nueva plataforma de adquisición y procesado de cantidades masivas de datos sobre medio ambiente a nivel global mediante el uso de técnicas Deep Learning para su posterior visualización.
El proyecto SKYDIPPER se centra en el desarrollo de una tecnología pionera, gracias a sus características de flexibilidad, escalabilidad e interoperabilidad para la toma de decisiones relacionadas con el medio ambiente en base a la visualización de los datos adquiridos y procesados.
